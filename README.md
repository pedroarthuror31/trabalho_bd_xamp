# trabalho_bd_xamp
O Sistema de Gerenciamento Acadêmico é uma aplicação web desenvolvida para a disciplina de Banco de Dados I com o objetivo de demonstrar a integração entre uma aplicação web e um banco de dados relacional. O sistema permite o gerenciamento de alunos, atividades e entregas, implementando operações CRUD completas para as entidades principais e utilizando uma modelagem relacional composta pelas tabelas alunos, atividades e aluno_atividade, responsável pelo relacionamento muitos-para-muitos entre alunos e atividades. A aplicação possibilita o cadastro de atividades para múltiplos alunos, o registro de notas e datas de entrega, pesquisas por filtros e consultas SQL utilizando SELECT, WHERE, INNER JOIN, NOT IN, ANY, EXISTS, além de funções de agregação como COUNT, AVG e GROUP BY. Também foram aplicadas restrições de integridade referencial, como PRIMARY KEY, FOREIGN KEY, UNIQUE, ON DELETE RESTRICT, ON DELETE CASCADE e ON UPDATE CASCADE, garantindo a consistência dos dados. Desenvolvido com PHP, HTML, CSS, MySQL e XAMPP, o projeto proporciona uma experiência prática no desenvolvimento de aplicações web integradas a bancos de dados relacionais, atendendo aos requisitos propostos para a disciplina de Banco de Dados I.
