# trabalho_bd_xamp
O Sistema de Gerenciamento Acadêmico é uma aplicação web desenvolvida para a disciplina de Banco de Dados I com o objetivo de demonstrar a integração entre uma aplicação web e um banco de dados relacional. O sistema permite o gerenciamento de alunos, atividades e entregas, implementando operações CRUD completas e utilizando uma modelagem relacional composta pelas tabelas alunos, atividades e aluno_atividade, que representa o relacionamento muitos-para-muitos entre alunos e atividades. Além disso, possibilita o registro de notas e datas de entrega, pesquisas e consultas utilizando filtros, INNER JOIN e subconsultas com NOT IN, bem como o uso de chaves primárias, chaves estrangeiras e regras de integridade referencial (RESTRICT e CASCADE). A aplicação foi desenvolvida utilizando PHP, HTML, CSS e MySQL, proporcionando uma experiência prática no desenvolvimento de aplicações web integradas a bancos de dados relacionais e atendendo aos requisitos propostos para a disciplina.
