# trabalho_bd_xamp
O Sistema de Gerenciamento Acadêmico é uma aplicação web desenvolvida para a disciplina de Banco de Dados I com o objetivo de demonstrar a integração entre uma aplicação web e um banco de dados relacional. O sistema permite o cadastro, consulta, edição e exclusão de alunos e atividades acadêmicas, implementando operações CRUD completas para ambas as tabelas. Além disso, oferece funcionalidades de pesquisa utilizando consultas SQL com filtros e relacionamentos entre tabelas por meio de chaves primárias e estrangeiras. A aplicação foi desenvolvida utilizando PHP, HTML, CSS e MySQL, aplicando conceitos de modelagem de dados, integridade referencial, consultas SQL e INNER JOIN para relacionar as atividades aos seus respectivos alunos. O projeto busca proporcionar uma experiência prática no desenvolvimento de aplicações web integradas a bancos de dados relacionais, atendendo aos requisitos propostos para a disciplina.
